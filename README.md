# Altamash Qureshi — Personal Portfolio

> **Live site:** [AltamashQureshi.github.io/portfolio](https://altamashqureshi.github.io/portfolio) *(deployed via GitHub Pages)*

A single-page personal portfolio for **Altamash Qureshi**, DevOps Engineer & Deputy Manager at Jio Platforms Limited. Built as a single `index.html` file with embedded CSS and JavaScript — no build tools or dependencies required.

---

## 📋 Sections

| Section | Description |
|---|---|
| **Hero** | Name, animated typewriter title, CTA buttons, profile photo |
| **About** | Professional summary, location, open-to-work status, key stats |
| **Skills** | Skill bars across CI/CD, Cloud, IaC, Monitoring, Security, and Scripting |
| **Experience** | Work history at Jio Platforms (DevOps Engineer & Deputy Manager) and prior consulting roles |
| **Certifications** | Build Automation, Elastic/ELK, Cisco Cybersecurity, AWS Migration, Packet Tracer |
| **Contact** | Email, phone, and LinkedIn links |

---

## ✨ Features

- **Dark, gold-accented theme** — custom CSS variables, smooth scroll, scroll-progress bar
- **Custom cursor** — animated dual-ring cursor (dot + ring)
- **Canvas particle background** — subtle animated particle network
- **Skill bars** — animated on scroll via IntersectionObserver
- **Animated counters** — years of experience, nodes managed, uptime SLA, deploy speed-up
- **Typewriter effect** — cycling job titles in the hero section
- **Scroll reveal animations** — elements fade/slide in as they enter the viewport
- **Fully responsive** — collapses gracefully on mobile/tablet
- **No external JS frameworks** — vanilla JavaScript only
- **Fonts via Google Fonts** — Playfair Display, DM Sans, JetBrains Mono

---

## 🚀 Deployment

The repo includes a **GitHub Actions workflow** (`.github/workflows/`) that automatically deploys the site to GitHub Pages on every push to `main`.

To run locally, simply open `index.html` in any modern browser — no server or build step needed.
